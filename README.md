# Joshua Sipos: ECE465 Final Project
This is the github repo to my final project for ECE 465. This project creates ten labs that are meant to be connected to the main course of 465 as if the labs were actually apart of the course.

## Contents

The repo contains the lab documents that student will use along with the answer keys that TAs will use for grading.

## Outline
1. Intro to circuitverse and circuit review
2. Boolen Minimization and Prime Implicants
3. Branch and Bound Optomization
4. Encoders
5. Priority Encoders and Decoders
6. Multiplexers
7. Intro to verilog
8. Finite State Machines
9. Synchronous Sequential Circuit Construction
10. Timing and Asynchronous Sequential Circuits
